# 时间线图 (Timeline)

## 图示说明
时间线图用于按时间顺序展示事件序列，适合展示历史发展、项目进程或事件编年。

## 适用范围
- 历史事件展示
- 项目发展历程
- 公司/产品里程碑
- 个人履历
- 项目复盘

## 语法示例

```mermaid
timeline
    title 项目发展历程

    2020年 : 项目启动
        完成可行性研究
        组建核心团队
    2021年 : 快速成长
        v1.0发布
        获得A轮融资
        团队扩展到50人
    2022年 : 稳步发展
        v2.0发布
        拓展海外市场
        获评行业领先
    2023年 : 创新突破
        v3.0发布
        推出AI功能
        用户突破1000万
```

```mermaid
timeline
    title 技术发展史

    1990年代 : 萌芽期
        互联网诞生
        Web 1.0时代
    2000年代 : 发展期
        Web 2.0兴起
        社交网络出现
    2010年代 : 爆发期
        移动互联网
        云计算普及
        大数据时代
    2020年代 : 智能化
        AI/ML崛起
        元宇宙概念
        AIGC爆发
```

## 语法说明

### 基本语法
```mermaid
timeline
    title 标题

    时间点1 : 事件1
        事件1详情
    时间点2 : 事件2
        事件2详情
        事件2额外说明
```

### 时间点格式
- 年份: `2020`
- 年月: `2020-03`
- 具体日期: `2020-03-15`
- 时代: `1990年代`

### 事件层级
- 主事件: 冒号后的第一行
- 子事件: 缩进的详细说明

### 多事件并行
```mermaid
timeline
    title 示例

    2023年 : 事件A
        : 事件B
        : 事件C
```

## 配置说明

### 样式选项
```mermaid
timeline
    title 示例
    sectionStyle 1 fill:#eaf
    sectionStyle 2 fill:#efe
```

### 主题支持
可以配合 Mermaid 主题配置颜色和样式。

### 注意事项
- 时间顺序很重要
- 事件数量适中
- 标签简洁明了
