# 状态图 (State Diagram)

## 图示说明
状态图用于描述一个对象或系统在其生命周期内所经历的各种状态，以及导致状态转换的事件或条件。

## 适用范围
- 对象状态管理设计
- 状态机建模
- 业务流程状态流转
- 协议状态机
- UI 组件状态管理

## 语法示例

```mermaid
stateDiagram
    [*] --> 初始状态
    初始状态 --> 处理中: 事件1
    处理中 --> 完成: 事件2
    处理中 --> 错误: 事件3
    错误 --> 处理中: 重试
    完成 --> [*]

    state 处理中 {
        [*] --> 步骤1
        步骤1 --> 步骤2
        步骤2 --> [*]
    }
```

```mermaid
stateDiagram-v2
    [*] --> 空闲

    空闲 --> 进行中: 开始任务
    进行中 --> 暂停: 暂停命令
    暂停 --> 进行中: 继续命令
    进行中 --> 完成: 任务完成

    state 完成 {
        [*] --> 保存结果
        保存结果 --> 生成报告
        生成报告 --> [*]
    }

    完成 --> [*]: 结束
```

## 语法说明

### 基本语法
```mermaid
stateDiagram
    [*] --> 状态名称: 转换条件
    状态名称 --> [*]: 结束转换
```

### 复合状态
```mermaid
stateDiagram
    state 复合状态名 {
        [*] --> 子状态1
        子状态1 --> 子状态2
        子状态2 --> [*]
    }
```

### 条件判断
```mermaid
stateDiagram
    状态A --> 状态B: 条件X
    状态A --> 状态C: 条件Y
```

### 特殊标记
- `[*]`: 开始或结束状态
- `state xxx`: 普通状态定义
- `state xxx { ... }`: 复合状态

### 历史状态
```mermaid
stateDiagram
    state 暂停状态 {
        H --> 恢复点
        [*] --> 恢复点
    }
```

## 配置说明

| 配置项 | 说明 |
|--------|------|
| showNullElements | 显示空元素 |
| hideEmptyDescription | 隐藏空描述 |
| fork/join | 分叉/汇合支持 |

### 样式
```mermaid
stateDiagram
    [*] --> 状态1
    状态1: 这是状态描述

    classDef active fill:#f96
    class 状态1 active
```
