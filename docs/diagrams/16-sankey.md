# Sankey (桑基图)

## 图示说明
桑基图是一种能量流图，用于展示数据在不同状态或类别之间的流动。常用于能源分析、成本分解、流量分析等场景。

## 适用范围
- 能源流向分析
- 成本结构分析
- 用户行为流程
- 预算分配展示
- 转化漏斗分析

## 语法示例

```mermaid
sankey-beta

    访问用户["访问用户<br>1000"]: 900
    注册用户["注册用户<br>300"]: 300
    活跃用户["活跃用户<br>150"]: 150
    付费用户["付费用户<br>50"]: 50

    访问用户 --> 注册用户: 300
    访问用户 --> 跳出用户["跳出用户<br>600"]: 600
    注册用户 --> 活跃用户: 150
    注册用户 --> 流失用户["流失用户<br>150"]: 150
    活跃用户 --> 付费用户: 50
    活跃用户 --> 免费用户["免费用户<br>100"]: 100
```

## 语法说明

### 基本语法
```mermaid
sankey-beta

    节点A["标签<br>数值"]: 流量值
    节点B["标签"]: 流量值

    节点A --> 节点B: 流量值
```

### 节点格式
```mermaid
sankey-beta
    节点名称["显示标签"]: 数值
```

### 节点命名规则
- 使用有意义的名称
- 可以包含 HTML 换行 `<br>`
- 数值表示该节点的流量大小

### 连接语法
```mermaid
sankey-beta
    A --> B: 流量值
    A --> C: 流量值
    B --> D: 流量值
```

### 流量计算
- 输入流量总和 = 输出流量总和
- 流量值应与连接线上的值匹配

## 配置说明

### Sankey 配置选项
```mermaid
sankey-beta
    config
        width: 800
        height: 600
```

### 样式设置
```mermaid
sankey-beta
    style 节点 fill:#f9f,stroke:#333
```

### 注意事项
- Sankey 是 beta 功能，语法可能变更
- 流量需要守恒（输入=输出）
- 节点和流量数量适中
- 建议在正式项目中使用前验证语法
